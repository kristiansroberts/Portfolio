# **Kristian Roberts Porfolio**
This is a simple, almost static portfolio site to host my resume and some some project details. 
Though Django was not necessarily needed, the website is built on Django, as I simply wanted some practice with it because I actually really enjoy it. The website is hosted on AWS using Elastic Beanstalk

### Stack
- Django 5
- SQLite
- Pillow
- pyhton-dotenv
- AWS for hosting

### Motivation
I wanted to work on a simple project which I could gain experience with making a full website from scratch, using Django, and more experience with using AWS.

### Contents
Currently, the website features in a home page, an about page, and a projects page.
